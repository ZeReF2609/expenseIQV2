# expenseIQ

expenseIQ is a smart expense tracking and management application built with Next.js. It helps users to register, classify, analyze and control their daily expenses with support for AI-powered features.

## Features

*   **Dashboard:** A comprehensive overview of your finances with KPIs and charts for total expenses, top categories, and transactions.
*   **Expense Tracking:** Manually add expenses or use the OCR feature to automatically scan and extract data from receipts.
*   **AI-Powered OCR:** Upload a receipt image, and the app will use AI to extract details like business name, date, total, and tax.
*   **Predictive Categorization:** The app uses AI to suggest an appropriate category for your new expenses based on their description.
*   **Budgets:** Set monthly budgets for different spending categories and track your progress.
*   **Custom Categories:** Create, manage, and customize your own expense categories with unique colors.
*   **Reports:** Analyze your spending habits with visual reports and date range filters.
*   **Customization:**
    *   **Theme Switching:** Choose between light and dark themes.
    *   **Language Selection:** The interface is available in English, Spanish, and Portuguese.

## Getting Started

To run the application locally, first install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:9002](http://localhost:9002) with your browser to see the result.
